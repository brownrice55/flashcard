# 単語帳

## 概要
<ul>
<li>単語を覚えては忘れ、覚えては忘れ、を繰り返す、それでも覚えたい、自分のためのアプリ。</li>
<li>趣味と勉強のための制作物です。少しずつブラッシュアップする予定です。</li>
</ul>
  
## 使い方
<ol>
<li>最初に10個以上単語を登録します。</li>
<li>ナビの「単語を覚える」を押して、自動再生かテストを選びます。</li>
<li>単語を覚えます。</li>
</ol>
